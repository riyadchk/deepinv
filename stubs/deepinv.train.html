train

deepinv.train(*args, **kwargs)[source]

Alias function for training a model using deepinv.training_utils.Trainer class.

This function creates a Trainer instance and returns the trained model.

Parameters:

  • args – Positional arguments to pass to Trainer constructor.

  • kwargs – Keyword arguments to pass to Trainer constructor.

Returns:

Trained model.
